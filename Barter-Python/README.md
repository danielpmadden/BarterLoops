
# Bartr MVP

## Quickstart
1. Install Docker Desktop and Git
2. Run:
   ```powershell
   .\run.ps1
   ```
   or on Mac/Linux:
   ```bash
   ./run.sh
   ```

## Features
- FastAPI backend with Postgres
- Frontend served with Nginx (static placeholder)
- Dockerized, one-click start

## Healthcheck
- API: http://localhost:8000/health
- Frontend: http://localhost
